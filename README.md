### Online Courses App built with Ruby on Rails

<hr>


##### Résumé des étapes : _Outils - Gems - Fonctionalités_
~~~ ~~~
- Création de la base de données

- Sqlite browser pour visualiser graphiquement la base de données

- Gem administrate + restriction d'accès à la base de données

- Gem paperclip : upload d'images pour présenter les cours

- DRY : utilisation de helpers + partials

- Gems : Bootstrap-saas + paramétrage de Turbolinks

- Création d'un helper pour personnaliser le titre des onglets des pages

- Ajout d'un model Episode pour les cours et d'une relation One To Many

- Gem devise pour la gestion des authentifications
